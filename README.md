# Abdulmalak Al Jamous - Portfolio

Professional portfolio for a Full Stack Software Engineer with EN/DE/AR localization, accessible UI, and production-ready SEO metadata.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router v6

## Features

- Multi-language support (EN/DE/AR) with RTL handling
- SEO-friendly metadata and structured data
- Responsive layout and accessible navigation
- Modular component architecture

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
